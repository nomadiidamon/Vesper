\doxylink{namespace_vesper}{Vesper} is a lightweight 2D/\+3D engine and editor inspired by The\+Cherno\textquotesingle{}s Game Engine Architecture series for the Hazel engine. ~\newline
 It provides a renderer, scene \& entity system, editor UI, input handling, and utilities to build simple games and interactive applications in C++ (C++17).

It is generally usable as an easily modifiable base line for creating graphics and software applications.

\begin{quote}
The specific purpose is for the creation of Particle System demos, experiments, and visualizations.

{\itshape {\bfseries{It is not intended to be a full-\/featured game engine,}}}

But rather an application to create a range of visual effects and particle system simulations. \end{quote}
