\doxysubsection{Detailed Description}
Base class for scriptable entities within a scene. 

A \doxylink{class_vesper_1_1_scriptable_entity}{Scriptable\+Entity} allows for custom behavior to be defined for entities in a scene, not already provided as standardized components.
